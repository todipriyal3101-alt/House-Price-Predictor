# 🏠 House Price Predictor

A machine learning web application that predicts house prices in Bangalore based on location, square footage, number of bedrooms (BHK), and bathrooms.

## 🚀 Live Demo
👉 [Click here to try the app](https://house-price-predictor-1-5aub.onrender.com)

## 🛠️ Tech Stack
- **Python**
- **Scikit-learn** — Ridge Regression model
- **Pandas & NumPy** — Data processing
- **Flask** — Web framework
- **HTML/CSS** — Frontend
- **Render** — Deployment

## 📊 Features
- Predicts house prices in **Bangalore**
- Select from **hundreds of locations**
- Input BHK, bathrooms, and square footage
- Instant price prediction in **Lakhs (₹)**

## 🧠 ML Model
- Algorithm: **Ridge Regression**
- Preprocessing: **OneHotEncoder** for location, **StandardScaler** for numerical features
- Trained on Bangalore house price dataset

## ⚙️ Run Locally
1. Clone the repo, install dependencies with `pip install -r requirements.txt`
2. Run `python main.py`
3. Open `http://localhost:5001`
